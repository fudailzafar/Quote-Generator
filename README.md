# Quote of the Day

A simple web application that fetches and displays a random quote along with its author. Users can get a new quote with a button click and share quotes on Twitter easily.

## Features

- Fetches random quotes from the Quotable API.
- Displays the quote along with the author's name.
- Option to share the quote on Twitter with a single click.

## File Structure

```bash
├── index.html               # Main HTML file
├── styles.css               # CSS file for styling
├── script.js                # JavaScript logic for fetching and displaying quotes
└── images/                  # Folder containing icons
    └── twitter.png          # Icon for the tweet button
```

## How to Use

1. Open the **index.html** file in your browser.
2. Click on the **New Quote** button to fetch a random quote.
3. Click on the **Tweet** button to share the quote on Twitter.
